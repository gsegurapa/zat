Airport Tracker Documentation

To display this help page: ?help or press the "Help" button in the configurator.

To open the app in a new window:

The airport tracker app is used to display flights departing from and arriving at a specified airport. It can also show weather images (including animated). The altitude of each airplane is indicated by the size of airplane (the higher the altitude, the larger the airplane) and by the shadow offset and fuzziness. Orange planes are arriving at the airport, while blue planes are departing. The smallest airplanes are taking off (blue) or landing (orange).

Configuration

The app can be configured using the interactive configuration tool. You can change any configuration parameter and see the results immediately.

Open the configuration tool by clicking on the Flightstats logo below the map. When open, you can drag the configuration tool around the map by dragging its title bar. To close the configuration tool, click on the "Hide" button, or click the Flightstats logo again.

Once you have configured the app, you can save your configuration by clicking on the "Save" button. Once saved, the app will always start up with your configuration. You can also reset the app back to the default configuration using the "Reset" button. Configurations are saved using a cookie on your local computer, so they are specific to that computer and browser.

Once you click the "Save" button and save the configuration, a URL is displayed. This URL can be used to start the app with the same configuration on a different computer or browser.

The "Help" button brings up this help page in a new window.

The "Full" button puts the airport tracker into full screen mode, and if pushed again, back to windowed mode. Full screen mode is not supported on all browsers, so this button may not appear. When in full screen mode the mouse cursor is hidden when over the map (you can still see it over the legend).

Weather Options

The app can optionally show ground-based weather radar from the US National Weather Service (US only). To show the current weather radar, set weatherFrames to "1". Alternatively, set weatherFrames between 2 and 25 to show an animation of the recent weather. The weather frames are updated approximately every 10 minutes.

If weatherStation is set to "automatic" then the app automatically finds the closest radar station to the specified airport. You can override this by specifying a station code. See the bottom of http://www.srh.noaa.gov/jetstream/doppler/ridge_download.htm for a list of stations and their locations. This can be useful if the closest station doesn't have the best weather coverage for the airport.

Browser Compatibility

The app has been tested on recent versions of Chrome, Firefox, Safari, and on IE 8 and 9. If you encounter any problems, please let us know.

There is a known issue with Safari that occasionally causes the airplanes to jiggle around on the screen (this might have been fixed in the most recent version of Safari).

Advanced/Experimental Options

These options must be specified in the URL, and are not available in the configurator and cannot be saved in a cookie.

interactive=true

Setting interactive to true allows you to drag the map using your mouse pointer and to zoom in by double clicking. It also enables the following keyboard shortcuts:

+ or =zoom in
– or _zoom out
arrow keysscroll the map by 100 pixels
space barscroll back to original position (with the airport centered)
return keyreturn to original scroll and zoom
Copen/close the configuration tool
Fenter/exit full screen (if supported by the browser)
Nwhen flip=true, this advances to the next airport
When interactive is true, the mouse cursor does not hide when in full screen mode.

thumb=true

Setting thumb to true puts the map into thumbnail mode. This is used to draw small maps (thumbnails). The main difference is that this option removes the legend from the bottom of the map.

jump=true

Setting jump to true eliminates smooth airplane animation so that the map uses far fewer processor resources. The airplanes jump from one position to another, and the labels are turned off. This option is often used with thumb=true.

flip=true

Setting flip to true changes the airport every minute. Airports are displayed from west to east, around the world. The current airport is displayed in the legend. You can also adjust the number of seconds between airport changes using the flipTime parameter. For example, flipTime=120 displays each airport for 2 minutes.

view=3D

Setting view to 3D displays the map in psuedo-3D. Note that this does not work in all browsers. It also sets interactive mode to true, so you can drag the map around in 3D.
